Atualização por USB

Guerrilha Pedais

Ferramenta de atualização de firmware. Tabela de partições opcional.

Browser incompatível. Isto precisa de Web Serial (normalmente Google Chrome ou Microsoft Edge no Mac ou no Windows). O Safari não suporta ligar a esta porta. Nenhum ficheiro sobe para a Internet — tudo acontece no teu computador.
A carregar a biblioteca de flash a partir da Internet (só a primeira abertura)…
Não deu a carregar o esptool.js.

Desactiva bloqueio de anúncios nesta página, experimenta janela anónima, outra rede (menos comum) ou o DevTools (F12) Consola para o erro. Primeiro CDN: jsdelivr.

1 Liga o dispositivo por USB
2 Clica Ligar e escolhe a porta
3 Escolhe o .bin e Gravar

Ligação

Se não ligar, tenta 115200 e desliga/religa o cabo.
